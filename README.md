# Using Locality Sensitive Hashing Based K-NN For Detecting Similar Product Reviews

This is a stating point for a larger project that focuses on applying many NLP algorithms and ML techniques to extract useful patterns from amazon product reviews.

## Getting Started

For getting this project running it is very recommended to use Google Research Colab.

### Prerequisites

- Python3
- NLP
- Some necessary terminal commands (unix)
